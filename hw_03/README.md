Консольное приложение, выполняющее сжатие данных двухпроходным алгоритмом Хаффмана.
 
Запуск:  
`cmake CMakeLists.txt`  
`make hw_03`   
`./hw_02 [option]...  [file]...`  

Флаги командной строки:
* `-c`: архивирование
* `-u`: разархивирование
* `-f`, `--file <путь>`: имя входного файла
* `-o`, `--output <путь>`: имя результирующего файла
